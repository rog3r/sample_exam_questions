 Features
 ----
 - Multiple choice questions support
 - Support images in questions and alternatives
 - Mobile-first

 ##### Manual installation

 Requirements are:

 Ruby 2.2
 Rails 4.2
 A server like Unicorn, Puma, Passenger or Thin


 Bundle the Gemfile

 ```sh
 bundle install
 ```


  Setup database
  ```sh
  rake db:setup
  ```
